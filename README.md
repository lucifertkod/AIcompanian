## 🚀 QUICK START (Even Your Grandma Can Do This)

```bash
# Clone the repo
git clone https://github.com/SuperShary/lonlybot.git

# Enter the den of loneliness
cd lonlybot

# Install dependencies (npm install your problems away)
npm install

# Run the thing
npm run dev

# Open http://localhost:3000 and say hi to your new girlfriend
```

### 🔑 **Getting Your API Key**

1. Go to [Google AI Studio](https://aistudio.google.com/)
2. Click "Get API Key"
3. Select a project (or create one)
4. Copy the key
5. Paste it in Settings → Intelligence → Secret Key
6. Click **Save & Connect**

That's it. You're now in a relationship. Congrats. 🎉
